# Faucet

The Qantera Faucet distributes QTER test tokens.

QTER is used to pay gas fees and test applications on the Qantera Public Testnet.

### Faucet information

| Field          | Value       |
| -------------- | ----------- |
| Token          | QTER        |
| Claim Amount   | `1 QTER`    |
| Faucet URL     | Coming soon |
| Claim Cooldown | 24 Hours    |

### How to claim

When the faucet becomes available:

1. Open the official faucet.
2. Connect your wallet.
3. Switch to Qantera Public Testnet.
4. Confirm chain ID `974621`.
5. Select **Claim 1 QTER**.
6. Wait for the transaction.
7. Check it on the [Qantera Explorer](https://explorer.qantera.network/).

### Safety

The faucet will never request:

* a seed phrase;
* a private key;
* payment;
* remote access;
* an unlimited token approval.

QTER on the Public Testnet has no monetary value.
