# Rabby and other EVM wallets

Qantera Public Testnet can be added to Rabby and other wallets that support custom EVM networks.

### Install Rabby Wallet

Download Rabby from the official website:

[Download Rabby Wallet](https://rabby.io/)

### Add Qantera to Rabby

1. Open Rabby Wallet.
2. Open the network management page.
3. Select **Add Custom Network**.
4. For the latest RPC endpoints, chain ID, native token, and explorer information, see [**Network Details**.](../introducing/network-details.md)
5. Save and switch to Qantera Public Testnet.

### Other EVM wallets

Use the same network information when adding Qantera to another compatible wallet.

Always verify:

* chain ID is `974621`;
* native token is `QTER`;
* RPC domain ends with `qantera.network`;
* explorer is [explorer.qantera.network](https://explorer.qantera.network/).

### Wallet compatibility

Standard EVM wallets can connect to the current Qantera Public Testnet.

Future quantum-resistant signing features may require additional wallet or signer support. Any new requirements will be announced before activation.

***

## Request Testnet Tokens

QTER is the native token of Qantera Public Testnet.

It is used to pay gas fees and test applications on the network.
