# Add Qantera to OKX Wallet

This guide explains how to connect OKX Wallet to Qantera Public Testnet.

### Install OKX Wallet

Download OKX Wallet from the official website:

[Download OKX Wallet](https://web3.okx.com/)

Use a separate wallet or account for testnet activities.

### Add Qantera manually

1. Open OKX Wallet.
2. Open the network selector.
3. Select **Manage Networks** or **Custom Network**.
4. Choose **Add Network**.
5. For the latest RPC endpoints, chain ID, native token, and explorer information, see [**Network Details**.](../introducing/network-details.md)
6. Save the network.
7. Switch to **Qantera Public Testnet**.

### Verify the connection

Before signing a transaction, confirm:

```
Chain ID: 974621
Token: QTER
```

Use the [Qantera Explorer](https://explorer.qantera.network/) to check your address and transactions.

### Backup RPC

If the primary endpoint is unavailable, use:

[https://rpc2.qantera.network](https://rpc2.qantera.network)

### Important

Connecting OKX Wallet does not automatically provide campaign points or rewards.

Any wallet multiplier or campaign benefit will be displayed on the official Qantera campaign page.
