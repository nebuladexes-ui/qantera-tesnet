# Overview

A node is software that connects to the Qantera network, verifies blockchain data, and shares information with other network participants.

Nodes help the network remain accessible, transparent, and independently verifiable.

### What a node does

A Qantera node may:

* connect to other network peers;
* receive and verify blocks;
* process network transactions;
* store blockchain data;
* provide RPC access to wallets and applications;
* support block explorers and indexers;
* participate in validation when configured as a validator.

### Current public infrastructure

Users and developers can currently connect through:

* [Primary RPC](https://rpc1.qantera.network)
* [Backup RPC](https://rpc2.qantera.network)
* `wss://rpc1.qantera.network`
* [Qantera Explorer](https://explorer.qantera.network/)

### Node participation status

The public Qantera node client has not yet been released.

The following information will be added when node participation becomes available:

* node software;
* installation guide;
* hardware requirements;
* genesis file;
* bootnodes;
* network ports;
* Docker image;
* synchronization options;
* snapshot downloads.

Do not download node software from unofficial sources.
