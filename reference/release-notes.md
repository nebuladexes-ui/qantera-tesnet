# Release notes

This page records important Qantera Public Testnet updates.

### Current Network Configuration

For the latest RPC endpoints, chain ID, native token, and explorer information, see [**Network Details**.](../introducing/network-details.md)

### Current Status

The following services are available:

* Public Testnet;
* HTTP RPC;
* backup HTTP RPC;
* WebSocket RPC;
* Block Explorer;
* EVM-compatible smart-contract deployment.

The following features are still being prepared:

* public faucet;
* public node client;
* validator onboarding;
* post-quantum technical specification;
* Genesis NFT mint;
* contributor leaderboard.

### Release Note Format

Future updates should use this structure:

#### Version and Date

```
Version:
Release Date:
Activation Block:
Required Node Version:
```

#### Changes

* protocol updates;
* RPC updates;
* smart-contract compatibility changes;
* wallet changes;
* explorer updates;
* faucet updates;
* security improvements.

#### Breaking Changes

List any changes that require users, developers, or node operators to update their configuration.

#### Known Issues

List active issues and temporary limitations.

#### Required Actions

Explain whether users or developers need to:

* update a wallet;
* change an RPC endpoint;
* update node software;
* redeploy an application;
* migrate an account;
* take no action.

> Qantera users should review release notes before major network upgrades.
