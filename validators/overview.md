# Overview

Validators help secure the Qantera network by participating in block validation and consensus.

They are responsible for operating reliable infrastructure and following the network protocol.

### Validator responsibilities

A validator may be responsible for:

* verifying transactions;
* validating blocks;
* participating in consensus;
* maintaining network availability;
* running the required node version;
* protecting validator keys;
* monitoring validator performance;
* completing protocol upgrades on time.
