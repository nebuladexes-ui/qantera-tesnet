# Requirements

Official validator requirements have not yet been announced.

### Expected requirement categories

The final guide will include:

#### Hardware

* CPU;
* memory;
* storage;
* bandwidth;
* supported operating systems.

#### Network

* stable internet connection;
* public IP requirements;
* required ports;
* peer connectivity;
* system time synchronization.

#### Validator eligibility

* permissionless, permissioned, or allowlisted access;
* staking requirements;
* testnet registration requirements;
* validator capacity;
* activation schedule.

#### Operational security

* protected validator keys;
* restricted server access;
* monitoring and alerts;
* backup and recovery procedures;
* upgrade readiness.

### Recommended preparation

Potential validator operators can prepare by learning:

* Linux server administration;
* firewall configuration;
* node monitoring;
* private-key security;
* infrastructure backups;
* incident response.

Do not purchase hardware specifically for Qantera until the official requirements are published.
