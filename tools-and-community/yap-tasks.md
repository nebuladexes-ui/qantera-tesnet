# Yap Tasks

Yap Tasks reward users for creating original content about Qantera.

### How it works

1. Connect your wallet.
2. Read the active content mission.
3. Create an original post on X.
4. Copy the post link.
5. Submit the link.
6. Wait for verification.
7. Receive points based on the result.

### Content may be evaluated by

* relevance;
* originality;
* clarity;
* technical accuracy;
* engagement;
* consistency.

### Content guidelines

Create content that:

* explains Qantera clearly;
* uses accurate network information;
* helps other users understand the project;
* avoids misleading claims;
* does not copy another contributor.

### Content that may be rejected

* copied posts;
* spam;
* unrelated content;
* deleted posts;
* fake engagement;
* misleading quantum-security claims;
* repeated submissions of the same content.

The active Yap Task page will display the current mission, point values, and submission limits.
