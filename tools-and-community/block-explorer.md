# Block explorer

The Qantera Explorer allows users to view activity on the Qantera Public Testnet.

Open:

[Qantera Explorer](https://explorer.qantera.network/)

### What you can search

You can search for:

* transaction hashes;
* wallet addresses;
* block numbers;
* contract addresses;
* tokens;
* NFTs, when indexed.

### Transaction information

A transaction page may display:

* status;
* sender;
* recipient;
* transferred amount;
* gas used;
* transaction fee;
* block number;
* transaction data.

### Address information

An address page may display:

* QTER balance;
* transaction history;
* token activity;
* contract interactions.

### Contract information

For smart contracts, the explorer may show:

* deployed bytecode;
* contract creator;
* deployment transaction;
* verified source code;
* read and write functions;
* contract events.
