# Tools overview

Qantera provides tools for exploring the network, receiving test tokens, testing applications, and participating in the community.

### Available tools

| Tool           | Purpose                                             |
| -------------- | --------------------------------------------------- |
| Block Explorer | View blocks, transactions, addresses, and contracts |
| Faucet         | Receive QTER test tokens                            |
| RPC            | Connect wallets and applications                    |
| Social Tasks   | Complete community activities                       |
| Yap Tasks      | Create original Qantera content                     |
| Genesis NFT    | Mint an early-contributor NFT                       |
| Leaderboard    | Track points and contributor rank                   |

### Official network tools

* [Qantera Explorer](https://explorer.qantera.network/)
* [Primary RPC](https://rpc1.qantera.network)
* [Backup RPC](https://rpc2.qantera.network)

Some community tools are still being prepared and will be linked when available.
