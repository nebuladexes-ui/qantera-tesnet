# Overview

Qantera is an EVM-compatible Layer 1 network that supports Solidity smart contracts.

Developers can use familiar Ethereum tools to build and test applications on the Qantera Public Testnet.

### What you can build

Examples include:

* tokens;
* NFTs;
* marketplaces;
* DeFi applications;
* on-chain games;
* governance systems;
* testnet utilities.

### Supported development tools

You can develop with:

* [Hardhat](https://hardhat.org/)
* [Foundry](https://getfoundry.sh/)
* [Remix](https://remix.ethereum.org/)
* [ethers](https://docs.ethers.org/)
* [viem](https://viem.sh/)
* [Web3.py](https://web3py.readthedocs.io/)

| Explorer     | [Qantera Explorer](https://explorer.qantera.network/) |

### Development flow

1. Write the contract.
2. Compile the contract.
3. Test it locally.
4. Connect to Qantera Public Testnet.
5. Deploy the contract.
6. Save the contract address.
7. Interact with the contract.
8. Verify the source code when verification becomes available.
