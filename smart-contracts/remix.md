# Remix

### After deployment

Copy the contract address and open:[Remix](https://remix.ethereum.org/) is a browser-based Solidity development environment.

It is useful for simple test contracts and first-time deployments.

### Deploy with Remix

1. Open [Remix](https://remix.ethereum.org/).
2. Create a new Solidity file.
3. Add your contract code.
4. Open **Solidity Compiler**.
5. Compile the contract.
6. Open **Deploy & Run Transactions**.
7. Select **Injected Provider**.
8. Connect your wallet.
9. Confirm the wallet is on Qantera Public Testnet.
10. Select **Deploy**.
11. Approve the transaction.

### Confirm the network

Before deployment, check:

```
Chain ID: 974621
Native Token: QTER
```
