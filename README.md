# What is the Qantera Testnet?

Qantera Public Testnet is an EVM-compatible Layer 1 network designed for the post-quantum era.

It provides a public environment where users and developers can test transactions, wallets, smart contracts, and applications before the Qantera mainnet launches.

### What can you do on Qantera?

On the Qantera Public Testnet, you can:

* connect an EVM-compatible wallet;
* receive QTER test tokens;
* send test transactions;
* deploy and interact with Solidity smart contracts;
* explore blocks, transactions, addresses, and contracts;
* connect applications through Qantera RPC endpoints.

### Quantum-resistant direction

Qantera is being developed with a focus on quantum-resistant transaction security and cryptographic agility.

Detailed information about the post-quantum algorithm and transaction-verification model will be published in the technical specification.
