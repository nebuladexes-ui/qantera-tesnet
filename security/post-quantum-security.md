# Post-quantum security

Qantera is being developed with a focus on quantum-resistant transaction security.

The goal is to prepare the network for a future where some classical cryptographic systems may no longer provide sufficient protection.

### Qantera's direction

Qantera is designed around:

* quantum-resistant transaction authentication;
* cryptographic agility;
* controlled security upgrades;
* compatibility with familiar Web3 applications.

### Current status

The final post-quantum specification has not yet been published.

Future technical documentation will explain:

* the signature algorithm;
* the selected parameter set;
* how transactions are signed;
* where signatures are verified;
* wallet requirements;
* account migration;
* upgrade procedures.

Until those details are released, Qantera does not make absolute claims such as:

* quantum-proof;
* unbreakable;
* impossible to hack.

### What post-quantum security does not prevent

Quantum-resistant cryptography does not protect users from:

* stolen seed phrases;
* phishing;
* malicious wallet extensions;
* fake websites;
* smart-contract vulnerabilities;
* compromised devices.

Users must still follow standard wallet and Web3 security practices.
