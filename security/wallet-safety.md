# Wallet safety

Your wallet controls your Qantera account.

Anyone with access to your seed phrase or private key can control the account.

### Basic safety rules

* Never share your seed phrase.
* Never share your private key.
* Use a separate wallet for testnet activity.
* Verify the network before signing.
* Review the recipient and amount.
* Use only official Qantera links.
* Avoid unknown wallet extensions.
* Do not install software from unofficial sources.

### Verify the network

Before signing, confirm:

```
Network: Qantera Public Testnet
Chain ID: 974621
Native Token: QTER
```

Official infrastructure:

* [Primary RPC](https://rpc1.qantera.network)
* [Backup RPC](https://rpc2.qantera.network)
* [Qantera Explorer](https://explorer.qantera.network/)

### Testnet wallet recommendation

Use a dedicated wallet that does not hold valuable assets.

QTER test tokens have no monetary value. Never pay anyone for testnet QTER.

### Common scams

Be careful with websites or messages that ask you to:

* import your wallet using a seed phrase;
* send funds before receiving test tokens;
* approve unlimited token access;
* install an unknown wallet plugin;
* provide remote access to your device.

Official Qantera support will never request your private key.
